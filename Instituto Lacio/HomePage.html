Eu estava aqui pesquisando a respeito do wordpress.org. Parece que é uma plataforma excelente para desenvolvimento de sites que pode reduzir bastante o tempo de produção.
Além de já ter temas prontos, é possível adicionar plugins que adicionam funcionalidades de todo o tipo para o site. Há um, por exemplo ( gratuito), que permite a criação de áreas exclusivas dentro do site, perfeito para áreas de assinante.
Há um outro que permite a criação de páginas modelo, sendo necessário, depois, somente acrescentar os dados
Cheguei a ver alguns que fornecem o ambiente EAD completo, mas são pagos, naturalmente.
A grande vantagem é que parece que o código é aberto e pode ser adaptado e alterado de acordo com a vontade do desenvolvedor.
vamos procurar mais a respeito disso. acho que vai dar para economizar um tempo fodido usando essa plataforma
